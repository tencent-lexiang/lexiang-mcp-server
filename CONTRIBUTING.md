# Contributing to Lexiang MCP Server

Thank you for your interest in contributing to the Lexiang MCP Server!

## Reporting Issues

If you encounter any issues:

1. **Lexiang Platform Issues**: Contact Lexiang support directly through the platform
2. **MCP Configuration Issues**: Open an issue in this repository with:
   - Your MCP client (CodeBuddy, Cursor, Claude Desktop, etc.)
   - Error messages you're seeing
   - Steps to reproduce the issue

## Feature Requests

We welcome feature requests! Please open an issue describing:
- The feature you'd like to see
- Your use case
- Any implementation ideas you have

## Documentation Improvements

Documentation improvements are always welcome. Feel free to submit pull requests for:
- Typo fixes
- Clarifications
- Additional examples
- Translations

## Code of Conduct

Please be respectful and constructive in all interactions. We're all here to make Lexiang MCP Server better!
