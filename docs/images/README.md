# Documentation Images

This directory contains images used in the documentation.

## Required Images

Please add the following images to this directory:

### 1. `lexiang-logo.png`
- The Lexiang logo
- Recommended size: 200x200px or similar square format
- Used in the README header

### 2. `mcp-demo.png` (optional)
- A screenshot or GIF showing the MCP server in action
- Shows AI assistant interacting with Lexiang knowledge base
- Recommended: animated GIF or clear screenshot

### 3. `mcp-config.png` (optional)
- Screenshot of the Lexiang MCP configuration page
- Shows where users can get their MCP URL

## Image Guidelines

- Use PNG format for logos and screenshots
- Use GIF format for animated demonstrations
- Keep file sizes reasonable (< 500KB for static images, < 2MB for GIFs)
- Ensure images are clear and readable
